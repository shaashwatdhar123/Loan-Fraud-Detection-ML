# Machine Learning-based Loan Fraud Detection

## Overview

Developed a machine learning pipeline to classify fraudulent loan applications using Support Vector Machines (SVM) and XGBoost. The project incorporates SMOTE for class imbalance handling, GridSearchCV for hyperparameter optimization, and 5-fold cross-validation to ensure robust model evaluation.

## Features

- Data preprocessing & one-hot encoding
- Feature scaling using StandardScaler
- SMOTE-based class balancing
- Support Vector Machine (SVM)
- XGBoost Classifier
- Hyperparameter tuning using GridSearchCV
- 5-Fold Cross Validation
- ROC-AUC based model evaluation
- Confusion Matrix & Classification Report
